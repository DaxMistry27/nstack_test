# Task Management Documentation

## Overview
This module handles task creation, retrieval, update, and deletion.

## Features
- Create task
- Get all tasks
- Get task by id
- Update task
- Delete task
- Filter by status and date
- Search by title and content
- Sort and paginate results

## Access Control
- Only authenticated users can access task routes.
- Users can access only their own tasks.

## Notes
- Task records are linked to the user who created them.
